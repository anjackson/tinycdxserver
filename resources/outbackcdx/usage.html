Wayback RemoteResourceIndex XML protocol

GET /[collection]?q=type:urlquery+url:[url]
Queries all available snapshots for the given URL. 
GET /[collection]?q=type:prefixquery+url:[urlPrefix]
Queries all records with the given URL prefix.

outbackcdx extensions

POST /[collection]
Adds records to the collection. Post body should be in 9 or 11 field CDX format. Creates the collection if it doesn't already exist. 
GET /[collection]?url=[url]
Plain text URL query.
